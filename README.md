# University Project Reports – Leiden University

A collection of project reports from courses I took at Leiden University. These are the ones I found most interesting or worth keeping.

## About

This repo is mainly for archiving and sharing the written side of my university work. The reports cover different topics depending on the course, but generally reflect how I approached problems, structured analyses, and presented results.


## No Code

I’m not including any code here.

Some of these projects are still used in courses, so sharing full solutions wouldn’t be appropriate. The focus of this repo is on the reports themselves rather than the implementations.

## Purpose

Mostly a personal archive, but also useful as a lightweight portfolio of my academic work.

## Disclaimer

These reports are shared for reference only. Don’t reuse or submit them as your own work. These reports sometimes showcase the download date of the document, not the submission date.
